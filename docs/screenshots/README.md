# Screenshots for documentation

Add setup and UI screenshots here so they can be referenced in CONTRIBUTING.md, README, or issues.

## Suggested screenshots

| File name         | What to capture |
|-------------------|-----------------|
| `01-clone-install.png` | Terminal after `git clone` and `npm install` (no errors). |
| `02-dev-server.png`    | Terminal with `npm run dev` and line showing `Local: http://localhost:3000`. |
| `03-browser-landing.png` | Browser open at http://localhost:3000 (landing page). |
| `04-browser-dashboard.png` | Browser open at a dashboard (e.g. `/dashboards/socialanalytics`). |

## How to take them

1. Run the steps in [CONTRIBUTING.md](../../CONTRIBUTING.md#running-the-project-on-your-machine).
2. At each step, capture the terminal or browser window.
3. Save images in this folder using the names above (or similar).
4. Reference them in markdown: `![Dev server](docs/screenshots/02-dev-server.png)`.
